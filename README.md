 SSI v0.0.1
===============
ShowSiteInfo brings various information to you, about the site which you want, through the following site:

- https://dnsbil.com

What do you need to use SSI ?
===================

- Python3.6
- Install `requirements.txt`

How to use SSI?
===================
Start it by typing:

    python3.6 dnsbil_console.py
